# MInecraft-Arcanacraft

太阳将要下山，在一片不断连接的山地里，一群人骑着马，拉着货车沿着官道走去。突然，一些面目可憎，许多肉已经腐烂的“人”拿着武器，向他们杀来。而他们有的拿出了武器，与怪物厮杀；有的则拿出了泛发奇异光泽的晶体，咏唱着什么，顷刻便是一片光明，让怪物们全身点燃。僵尸的袭击被打退，他们继续前进。

这是一个无数的农民用汗水供养着少数高贵者的不公正时代，也是一个刀光剑影闪耀的时代，一个神明高坐天穹的时代，一个奥术的时代。

在奥术的时代冒险吧，希望你能够在此找到乐趣。

本TRPG规则书是依据Mojang发行的游戏Minecraft及其模组以及Minecraft：Dungeons写作的。

MCwiki和MC百科为本规则书的写作提供了详实的参考资料。

《我的世界：奥术时代》企划的版主欧阳狮子与参与者一同写作了此规则书。
